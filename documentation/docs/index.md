# Main

Welcome to the documentation of the project. Here is written the plan of the project. The project is divided into 5 phases. Each phase will have a set of features that will be implemented. The project will be written in both Python and C++. The Python code will be used for the high-level operations and the C++ code will be used for the low-level operations to achieve better performance. The C++ code will be binded to Python using `pybind11`.

## Test

The project will be tested using `pytest` for Python code and `googletest` for C++ code. The tests will be written for both languages. The C++ code will be binded to Python using `pybind11`. The tests will be written for the high-level operations in Python and the low-level operations in C++. The tests will be written for the core features of the project.

You can see the [Test Plan](Test.md) of the project.
